🔧 TROUBLESHOOTING RAPIDO
──────────────────────────────────────────────────────────────

PROBLEMA: "Copia IP non funziona"
SOLUZIONE: Apri con server HTTP (localhost:8000)
          Non usare file:// direttamente

PROBLEMA: CSS non si carica
SOLUZIONE: style.css deve essere nella stessa cartella

PROBLEMA: Menu mobile non appare
SOLUZIONE: Controlla console (F12) per errori JS

PROBLEMA: Animazioni lente
SOLUZIONE: Aggiorna il browser alla versione recente

PROBLEMA: Immagini non caricate
SOLUZIONE: Verifica percorso /images e formato
