⚜️ IMPERIALCRAFT - GUIDA CONFIGURAZIONE ⚜️

🚀 Primi Passi

Benvenuto nel setup di ImperialCraft! Segui questa guida per configurare il sito secondo le tue esigenze.

1. Struttura dei File

Assicurati di avere questi file nella stessa cartella:

sito2/
├── index.html
├── style.css
├── script.js
└── README.md (opzionale)

2. Apertura Locale

Per testare il sito localmente:

  1. Apri una riga di comando nella cartella del progetto
  2. Esegui: python -m http.server 8000 (Python 3)
  3. O con Python 2: python -m SimpleHTTPServer 8000
  4. Accedi a: http://localhost:8000
⚠️ IMPORTANTE: Apri sempre il file tramite server HTTP, non con file:// direttamente, per evitare problemi con CORS e funzionalità JavaScript.

🚀 Deployment Online

Opzione 1: GitHub Pages (Gratis)

  1. Crea un repository su GitHub
  2. Carica i file del sito
  3. Vai a Settings → Pages
  4. Seleziona "Deploy from branch" e "main"
  5. Salva e aspetta 1-2 minuti
  6. Il sito sarà disponibile su: https://tuousername.github.io/nome-repo

Opzione 2: Netlify (Gratis + Facile)

  1. Vai su netlify.com
  2. Fai login con GitHub
  3. Seleziona "Import an existing project"
  4. Seleziona il repository
  5. Deploy automatico istantaneo!

Opzione 3: Vercel (Gratis + Ultra-Fast)

  1. Vai su vercel.com
  2. Fai login con GitHub
  3. Importa il progetto
  4. Deploy automatico in pochi secondi
✅ CONSIGLIO: Per i migliori risultati, usa Netlify o Vercel. Sono gratuiti, veloci e offrono certificati SSL automatici.

🔧 Troubleshooting

Problema: "Copia IP" non funziona

Soluzione: Controlla che il file sia aperto tramite HTTP (non file://). Usa un server locale.

Problema: Menu mobile non appare

Soluzione: Verifica che script.js sia caricato correttamente. Controlla la console (F12) per errori JavaScript.

Problema: CSS non si carica

Soluzione: Assicurati che style.css sia nella stessa cartella di index.html. Controlla i percorsi nei link.

Problema: Animazioni non funzionano

Soluzione: Il tuo browser potrebbe non supportare CSS animations. Aggiorna il browser.
